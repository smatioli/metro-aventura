# ADR 0002: Controles compreensíveis sem leitura

- Status: aceito
- Data: 2026-07-21

## Contexto

O jogador principal ainda não sabe ler. Instruções escritas, inclusive nomes de teclas, não podem ser necessárias para jogar.

## Decisão

Cada ação será ensinada visualmente. A tela mostrará uma representação grande e reconhecível da tecla física, junto de uma animação curta demonstrando o resultado esperado.

As atividades usarão o menor número possível de teclas. O mesmo comando deverá manter o mesmo significado durante toda a experiência. Texto e nomes de estações serão informação complementar para adultos, nunca um requisito para avançar.

## Consequências

- Tutoriais serão demonstrativos, não textuais.
- Teclas ativas terão grande contraste e indicação visual suave.
- Toda instrução deverá ser validada sem som, pois áudio também não pode ser a única pista.
- Atividades baseadas em leitura ou escrita estão fora do escopo.
- Ainda é necessário escolher o dispositivo principal e o conjunto exato de controles.
