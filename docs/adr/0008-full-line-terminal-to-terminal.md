# ADR 0008: Viagem completa entre terminais

- Status: aceito
- Data: 2026-07-21

## Contexto

Permitir a escolha de qualquer estação de partida acrescentaria um menu extenso e dependente de muitos nomes. O jogador ainda não lê e a experiência deve permanecer simples.

## Decisão

Depois de escolher uma linha, o jogador escolherá um dos dois sentidos por meio de dois trens apontando em direções opostas. O nome do terminal de destino será falado quando cada opção for destacada.

A viagem começará em um terminal e seguirá a ordem completa das estações até o terminal oposto.

Ao final, haverá uma celebração visual curta e calma. O jogador poderá repetir a viagem ou voltar à seleção de linha usando setas e barra de espaço.

## Consequências

- A sequência de estações é sempre real e previsível.
- O progresso pode ser mostrado como uma linha simples de círculos, sem exigir leitura.
- Cada linha oferece duas viagens, uma em cada sentido.
- A seleção de uma estação inicial intermediária fica fora da primeira versão.
- A celebração não terá flashes, ruídos repentinos ou avanço automático.
