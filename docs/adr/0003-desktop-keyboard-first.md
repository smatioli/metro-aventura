# ADR 0003: Computador com teclado como plataforma principal

- Status: aceito
- Data: 2026-07-21

## Contexto

O jogador usará um computador. As instruções devem funcionar sem leitura e demonstrar visualmente as teclas necessárias.

## Decisão

O jogo será executado em uma página web para computador e terá o teclado físico como entrada principal. Os desenhos na tela reproduzirão o formato e a posição relativa das teclas reais.

O mouse poderá funcionar como alternativa para um adulto auxiliar, mas nenhuma atividade dependerá dele.

## Consequências

- O foco de interação deverá permanecer visível em tela cheia e após cliques acidentais.
- O navegador não poderá executar ações inesperadas quando as teclas do jogo forem usadas.
- Os comandos escolhidos devem ser grandes, fáceis de localizar e consistentes.
- O conjunto definitivo de teclas ainda precisa ser definido conforme a coordenação motora do jogador.
